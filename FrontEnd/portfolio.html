Projects

Project 1: Web Development Portfolio

Designed and developed a personal portfolio website using HTML, CSS, and JavaScript to showcase my web development skills. The site features a responsive design, an interactive project gallery with hover effects, and a contact form with validation. It highlights my ability to create visually appealing and functional web interfaces.

Project 2: Mobile Weather App

Developed a cross-platform mobile weather application using React Native, providing real-time weather updates and forecasts based on the user's location. Key features include current weather conditions, a 7-day forecast, location-based updates using GPS, and customizable weather alerts, demonstrating proficiency in mobile app development and API integration.

Project 4: Task Management Tool

Created a task management tool for team collaboration, allowing users to create, assign, and track tasks within projects. Features include user registration and login, project dashboards with progress indicators, and real-time notifications for task updates. The application was built using Angular for the frontend, Node.js and Express for the backend, and MongoDB for the database.
